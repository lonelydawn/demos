# counter

> 一个使用 Vue-cli 、 Vue 和 Vuex 的简单案例
>
> 内容：当点击按钮时，计数器自增 1

## 开始

##### 安装依赖
可以使用以下命令安装依赖：

```shell
 npm install
```

##### 开发

运行

```shell
 npm start
```

启动 webpack 服务，以调试项目。

##### 生产

如果要发布生产版本，运行

```shell
 npm run build
```

以编译和压缩源代码。

之后运行

```shell
 node app.js
```

启动一个 [koa](https://github.com/koajs/koa) 搭建的服务端。
